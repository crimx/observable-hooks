# 安装

observable-hooks 需要与 rxjs 和 react 一起工作。如项目没有需要一并安装。

yarn

yarn add observable-hooks rxjs react

npm

npm install --save observable-hooks rxjs react

# 使用

所有可用的 API 都能从主入口导入。

import { ... } from 'observable-hooks'
